Проект был выполнен для «Бета-Банка». Компании было необходимо сохранить текущих клиентов. Задачей проекта было построить эффективную модель для прогнозирования оттока клиентов «Бета-Банка».
В нашем распоряжении были данные о поведении клиентов «Бета-Банка», а также данные о расторжении договора с банком. Модель должна была достаточно точно предсказывать уйдёт клиент или нет.
В качестве метрики модели была выбрана F-1 мера, а также AUC-ROC.

Мы провели исследование задачи на 3-х разных моделях с учётом и без учёта дисбаланса классов. При обучении модели без учёта дисбаланса классов ни одна из моделей не показала достаточно хорошего результата.
Для борьбы с дисбалансом для каждой модели мы использовали три метода: встроенный аргумент `class_weight`, upsampling и downsampling.
Для модели логистической регрессии мы увеличили пороговую вероятность положительного класса, чтобы увеличить точность.

Лучший результат показала модель случайного леса с параметром  `class_weight='balanced'`: F1-мера - 0.61. AUC-ROC данной модели также была самой высокой - 0.83, то есть при высоком TPR (True positive rate), FPR достаточно низкий.
На тестовой выборке модель также получила высокое значение F1-меры: 0.6 и  AUC-ROC: 0.83.
